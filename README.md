# CUDA100

This repository contains the code developed during my 100 days of CUDA programming challenge.

0. [hello_world](./000_hello_world): test program to verify that the GPU is working correctly.
1. [vector_add](./001_vector_add): adds two vectors using the GPU.
2. [matrix_multiply](./002_matrix_multiply): multiplies two matrices using the GPU.
3. [matrix_transpose](./003_matrix_transpose): transposes a matrix using the GPU.
4. [color_inversion](./004_color_inversion): inverts the color of an image using the GPU.
5. [matrix_addition](./005_matrix_addition): adds two matrices using the GPU.
6. [conv1d](./006_conv1d): performs a 1D convolution using the GPU.
7. [reverse_array](./007_reverse_array): reverses an array using the GPU.
8. [relu](./008_relu): performs a ReLU activation using the GPU.
9. [rainbow_table](./009_rainbow_table): performs a rainbow table attack using the GPU.
10. [matrix_copy](./010_matrix_copy): copies a matrix using the GPU.
11. [count_array_elements](./011_count_array_elements): counts the number of elements in an array that are equal to a given value.
12. [count_array2d_elements](./012_count_array2d_elements): counts the number of elements in a 2D array that are equal to a given value.
13. [silu](./013_silu): performs a SiLU activation using the GPU.
